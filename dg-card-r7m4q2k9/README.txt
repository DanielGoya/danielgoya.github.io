INSTALACIÓN EN EL TELÉFONO
--------------------------
1. Sube la carpeta y espera a que GitHub Pages publique los cambios.
2. Abre esta dirección en el Samsung:
   https://danielgoya.github.io/dg-card-r7m4q2k9/
3. En Samsung Internet o Chrome, usa la opción para instalar la aplicación o
   agregarla a la pantalla de inicio. El nombre será “Daniel Goya”.
4. Abre el panel Edge, entra a Editar y agrega la nueva aplicación.
5. Al abrirla desde el panel Edge debería aparecer sin la interfaz normal del
   navegador y funcionar también sin conexión.

USO
---
- Toca el QR para ampliarlo a casi toda la pantalla.
- El botón de pantalla completa puede ocultar controles adicionales del navegador.
- La aplicación intenta mantener la pantalla encendida mientras se usa.
- Cuando otra persona abre la URL en un navegador, verá botones para guardar el
  contacto, abrir WhatsApp, escribir por correo y visitar el sitio.
- El teléfono personal es el número preferido de la vCard:
  +56 9 8844 8384.

PRIVACIDAD: LIMITACIÓN IMPORTANTE
---------------------------------
GitHub Pages no hace privada esta carpeta. Esta versión es DISCRETA o “NO LISTADA”:

- usa una dirección difícil de adivinar;
- no está enlazada desde las demás páginas;
- incluye instrucciones noindex para buscadores;
- no muestra el número personal como texto visible en la pantalla.

Sin embargo, cualquier persona que conozca la URL, escanee el QR o revise los
archivos del repositorio público puede acceder al contenido y obtener el número.
No uses esta solución para información verdaderamente confidencial.

Una página realmente privada requeriría alojamiento con autenticación. Eso sería
incompatible con que cualquier contacto pueda escanear el QR y abrir la tarjeta
sin iniciar sesión.
