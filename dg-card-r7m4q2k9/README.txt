TARJETA PWA DE DANIEL GOYA — VERSIÓN 7
==========================================

FUNCIÓN
-------
Esta PWA sirve para que Daniel muestre su identidad y el QR que conduce a:
https://danielgoya.github.io/contacto.html

El recuadro danielgoya.github.io es funcional y abre el sitio en una pestaña o
ventana nueva.

INSTALACIÓN
-----------
1. Sube la carpeta completa “dg-card-r7m4q2k9” a la raíz del repositorio
   danielgoya.github.io.
2. Espera uno o dos minutos para que GitHub Pages publique los cambios.
3. Abre en el teléfono:
   https://danielgoya.github.io/dg-card-r7m4q2k9/
4. Usa Samsung Internet o Google Chrome.
5. Cuando aparezca “Instalar tarjeta en este teléfono”, toca el botón y confirma.
6. Si no aparece:
   - Samsung Internet: menú ≡ → Añadir página a → Pantalla de inicio.
   - Chrome: menú ⋮ → Instalar aplicación o Añadir a pantalla principal.
7. Abre la aplicación instalada una vez.
8. Para añadirla al panel Edge:
   - abre el panel Edge;
   - toca Editar;
   - busca “Daniel Goya”;
   - agrégala al panel.

IDIOMA EN LA PWA
----------------
La selección inicial NO utiliza navigator.language ni las preferencias de idioma
configuradas en tu teléfono.

Utiliza la zona horaria accesible mediante el navegador:
- español si la zona horaria está asociada a un país o territorio
  hispanohablante;
- inglés para cualquier otra zona;
- inglés si no se puede leer la zona horaria.

El control ES/EN permanece disponible. Se puede pulsar cualquier parte del
control, no solamente las letras.

No se solicita geolocalización: el permiso persistente no es necesario para este
uso y la zona horaria permite la selección automática sin mostrar una alerta de
ubicación al abrir la tarjeta.

ZOOM
----
Al tocar el QR se abre una vista de pantalla completa con:
- fotografía;
- nombre;
- Economista / Economist;
- enlace funcional a danielgoya.github.io;
- QR considerablemente mayor.

Para cerrar la vista ampliada, toca el fondo exterior.

ACTUALIZACIÓN
-------------
Si el teléfono sigue mostrando una versión anterior:
1. cierra completamente la PWA;
2. abre https://danielgoya.github.io/dg-card-r7m4q2k9/ en el navegador;
3. recarga;
4. vuelve a abrir la PWA;
5. si persiste, desinstálala e instálala otra vez.
